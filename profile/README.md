## Tools & Resources

### VS Code Extension
**[MDX SEO Validator](https://github.com/rampify-dev/mdx-seo-validator)** - Real-time SEO validation for MDX/Markdown files with live Google search preview. Validates titles, descriptions, heading hierarchy, and more.

- 500+ installs on VS Code Marketplace
- Next.js, Astro, and Remix support
- [Install from Marketplace →](https://marketplace.visualstudio.com/items?itemName=rampify.mdx-seo-validator)

### n8n Workflows
**[Rampify Workflows](https://github.com/rampify-dev/rampify-workflows)** - Automated SEO monitoring workflows for n8n. Daily site analysis with automated GitHub issue creation.

- Works with n8n Cloud and self-hosted
- Flexible notifications (GitHub, Slack, Email, Discord)
- [Get workflow →](https://github.com/rampify-dev/rampify-workflows)

### MCP Server
**[Rampify MCP](https://github.com/rampify-dev/rampify-mcp)** - Model Context Protocol server for Claude Desktop. Access SEO intelligence directly in your AI coding assistant.

- 6 operational tools (issues, page SEO, meta generation, schema generation, GSC insights)
- Works with Claude Desktop and other MCP clients
- [Install from npm →](https://www.npmjs.com/package/@rampify/mcp-server)

## Platform

[Rampify](https://rampify.dev) crawls your live site and turns it into structured SEO intelligence. Get insights about indexing, redirects, broken links, schema detection, and more — backed by real crawl data.

- Google Search Console integration
- API key authentication for MCP and automation
- Action plan with prioritized fixes
- Real-time monitoring and alerts
